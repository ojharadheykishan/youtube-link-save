## How to Use:
1. Login to VideoHub
2. Click "Telegram Study" in sidebar (opens in new tab)
3. Enter channel username (e.g., @channel_name)
4. Enter API ID and API Hash from my.telegram.org
5. Click "Fetch Videos" to auto-load videos
6. Select a folder and click "Save" to save videos to library
